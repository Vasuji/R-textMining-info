- 1. Startwith
```
^i think
```
- 2. At the end
```
morning$
```
- 3. Both upper and lower case
```
[Bb][Uu][Hh]
```
- 4. Both allowed at begining of line
```
^[Ii] am
```
- 5. Character range
```
^[0-9][a-zA-Z],^[a-z],^[0-9] etc.
```
- 6. Character Class with [],ending not with "?" or "."
```
[^?.]$
```
- 7. Any character
```
"."
```
